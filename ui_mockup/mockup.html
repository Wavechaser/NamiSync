About NamiSync
Color palette — the Fluent 2 token set this mockup is built from (see the :root block in this file's <style>). --mica is the one deliberately-tinted surface (title bar + tabs column); the main pane is a solid, neutral --card-* card floating on top of it. The host page around the window uses its own plain --host-* grays, unrelated to either. Accent and kind/status colors are carried over unchanged from the app's existing theme.
